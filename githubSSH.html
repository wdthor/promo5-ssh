Vérifier si les clés SSH existent déjà

Avant de générer une clé SSH, il faut vérifier qu'il n'en existe pas déjà un.

Pour vérifier les clés SSH existant

Ouvrer le terminal et saisissé:


            ls -al ~/.ssh

Vérifier dans la liste si il existe déjà une clé SSH publique

Par défault, les nom de fichier des clés publiques sont:


            id_dsa.pub
            id_ecdsa.pub
            id_ed25519.pub
Les autres selecteurs sont: 

            document.getElementById("monId");

selectionne un element par son id


            document.getElementsByClassName("mesClasses");

selectionne les elements par leur nom de classe


            document.getElementsByTagName("mesElement");

selectionne les elements HTML (ex: tous les li)

Pourquoi vous montrer ses selecteurs alors que l'on peut faire la même chose avec querySelector et querySelectorAll ?

Tous simplement car vous verrez souvent ses types de selecteurs dans la doc et dans le code d'autres personnes en ligne alors qu'ils sont simplement d'autres types de selecteurs.
